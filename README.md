# rescript-template
A template for our new lord ReScript

## prerequisites

Install Node.js from [here](https://nodejs.org/en/download/)

## how to use

```
npm run link // if you already have bs-platform installed

npm install // otherwise

npm run build
node src/Main.bs.js
```
